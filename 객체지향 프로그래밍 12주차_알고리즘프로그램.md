# 프로그램 코드와 설명

## 1. 해당 코드는 알고리즘 중 count에 대해 알아볼 수 있는 예제이다.

```cpp
#include <iostream>
#include <vector>
#include <algorithm> // count

using namespace std;

int main() {
    vector<int> vect = { 1, 2, 3, 2, 2, 4, 5 };

    int iVal = 2;

    int cnt = count(vect.begin(), vect.end(), iVal);

    cout << iVal << "의 갯수: " << cnt << endl;

    return 0;
}
```
결과: 
2의 갯수: 3

-> count는 특정한 값의 갯수를 찾아준다.

## 3. 해당 코드는 알고리즘 중 unique에 대해 알아볼 수 있는 예제이다.

```cpp
#include <iostream>
#include <vector>
#include <algorithm> 

using namespace std;

void PrintVector(vector<int> vect) {
    vector<int>::iterator it;

    it = vect.begin();

    while (it != vect.end()) {
        cout << *it << "\n";
        it++;
    }
}

int main() {
    vector<int> vect = { 1, 1, 2, 2, 2, 3, 3, 4 };

    cout << "unique전" << endl;
    PrintVector(vect);
    cout << endl;

    // 새 끝지점이 반환된다
    auto aNewEnd = unique(vect.begin(), vect.end());

    cout << "unique후" << endl;
    PrintVector(vect);
    cout << endl;

    cout << "실제로 바뀐 유효한 구간" << endl;
    for (auto it = vect.begin(); it != aNewEnd; it++) {
        cout << *it << " ";
    }

    return 0;
}
```
결과:
unique전
1
1
2
2
2
3
3
4

unique후
1
2
3
4
2
3
3
4

실제로 바뀐 유효한 구간
1 2 3 4

-> 결과를 통해 uniqe는 중복을 제거하고, 유효한 범위를 포인터로 반환한다. 그 포인터를 끝으로 기준잡고 반복문으로 확인시 유효한 구간만 출력되는걸 확인할 수 있다.
  

## 4. 해당 코드는 알고리즘 중 fill에 대해 알아볼 수 있는 예제이다.

```cpp
#include <iostream>
#include <vector>
#include <algorithm> // fill
using namespace std;

void PrintVector(vector<int> vect) {
    vector<int>::iterator it;

    it = vect.begin();

    while (it != vect.end()) {
        cout << *it << "\n";
        it++;
    }
}

int main() {
    vector<int> vect(5); 

    cout << "fill사용 전" << endl;
    PrintVector(vect);
    cout << endl;

    // 전체를 7로 채우기
    fill(vect.begin(), vect.end(), 7);

    cout << "fill사용 후" << endl;
    PrintVector(vect);
    cout << endl;

    return 0;
}
```

결과:
fill사용 전
0
0
0
0
0

fill사용 후
7
7
7
7
7

-> 결과를 통해 fill은 특정 값을 특정한 범위에 채운다.

## 5. 해당 코드는 알고리즘 중 replace에 대해 알아볼 수 있는 예제이다.

```cpp
#include <iostream>
#include <vector>
#include <algorithm> // replace

using namespace std;

void PrintVector(vector<int> vect) {
    vector<int>::iterator it;

    it = vect.begin();

    while (it != vect.end()) {
        cout << *it << "\n";
        it++;
    }
}

int main() {
    vector<int> vect = { 1, 2, 3, 2, 4, 2, 5 };

    cout << "replace 전" << endl;
    PrintVector(vect);
    cout << endl;

    // 벡터 안의 2를 전부 7로 변경
    replace(vect.begin(), vect.end(), 2, 7);

    cout << "[replace 후" << endl;
    PrintVector(vect);
    cout << endl;

    return 0;
}
```

결과:
replace 전
1
2
3
2
4
2
5

replace 후
1
7
3
7
4
7
5

-> 결과를 통해 replace는 특정 값을 다른 값으로 변환해준다.
