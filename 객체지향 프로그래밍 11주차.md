# 프로그램 코드와 설명

## 1. 해당 코드는 vector컨테이너의 개념을 알아보는 간단한 예제이다. push_back함수를 통하여 vector에 값을 추가하고,vector가 가진 반복자 begin과 end를 통해 원소 값을 출력한다.

```cpp
// 2025-11-11.cpp : 이 파일에는 'main' 함수가 포함됩니다. 거기서 프로그램 실행이 시작되고 종료됩니다.
//

#include <iostream>
#include <vector>
using namespace std;

int main()
{
   vector<int> vect;
	
   for (int i = 0; i < 6; i++) {
      vect.push_back(2 * i);
   }

   vector<int>::iterator it;

   it = vect.begin();

   while (it != vect.end()) {
      cout << *it << "\n";
      it++;
   }
}
```
결과:
0
2
4
6
8
10

-> 결과를 통해 vector는 배열과 비슷하지만 push_back을 통해 요소를 추가할 수 있고, 반복자를 통해 수월하게 출력이 가능하다.



## 2. 해당 코드는 set, multiset컨테이너의 개념을 알아보는 간단한 예제이다. insert를 통해 값을 추가하고, 반복자 begin과 end를 통해 원소 값을 출력한다.

```cpp
// 2025-11-11.cpp : 이 파일에는 'main' 함수가 포함됩니다. 거기서 프로그램 실행이 시작되고 종료됩니다.
//

#include <iostream>
#include <vector>
#include <set>
using namespace std;

int main()
{
   multiset<int> myset;

   myset.insert(7);
   myset.insert(2);
   myset.insert(1);
   myset.insert(-4);
   myset.insert(8);
   myset.insert(6);
   myset.insert(3);
   myset.insert(6);

   // set에서는 중복된 숫자를 하나로 처리한다.
   // multiset은 중복 숫자도 출력이 된다.
   //set<int>::iterator it;
   multiset<int>::iterator it;

   it = myset.begin();

   while (it != myset.end()) {

      cout << *it << endl;

      it++;
   }

}
```
결과1 set사용:

1
2
3
6
7
8

결과2 multiset사용:
-4

2

-> 결과를 통해 set과 multiset 두 컨테이너 다 insert된 값들을 정렬한다는 특징이 있지만 set은 중복을 출력하지 않고, multiset은 출력한다는 차이점이 존재한다.



## 2. 해당 코드는 map, multimap컨테이너의 개념을 알아보는 간단한 예제이다. insert를 통해 값을 추가하고, 반복자 begin과 end를 통해 원소 값을 출력한다. map은 출력시 어떤요소를 출력할때 명시해줘야한다(->)

```cpp
// 2025-11-11.cpp : 이 파일에는 'main' 함수가 포함됩니다. 거기서 프로그램 실행이 시작되고 종료됩니다.
//

#include <iostream>
#include <map>
#include <string>
using namespace std;

int main()
{
   multimap<string, string> mymap;

   // 첫번째값을 기준으로 정렬된다
   mymap.insert(make_pair("e", "apple"     ));
   mymap.insert(make_pair("c", "mango"     ));
   mymap.insert(make_pair("d", "peach"     ));
   mymap.insert(make_pair("d", "yummy"     ));
   mymap.insert(make_pair("a", "banana"    ));
   mymap.insert(make_pair("b", "pineapple" ));
   mymap.insert(make_pair("f", "pinepeach" ));

   // map에서는 중복된 숫자를 하나로 처리한다.
   // multimap은 중복 숫자도 출력이 된다.
   //map<>
   multimap<string, string>::iterator it;

   it = mymap.begin();

   while (it != mymap.end()) {

      cout << it->first << " " << it->second << endl;

      it++;
   }
}

```
결과1 set사용:
a banana
b pineapple
c mango
d peach
e apple
f pinepeach

결과2 multiset사용:
a banana
b pineapple
c mango
d peach
d yummy
e apple
f pinepeach

-> 결과를 통해 map과 multimap 두 컨테이너 다 insert된 값들을 정렬한다는 특징이 있지만 map은 중복을 출력하지 않고, multimap은 출력한다는 차이점이 존재한다.
